1.https://mariadb.org/download/?t=mariadb&p=mariadb&r=11.0.4&os=windows&cpu=x86_64&pkg=msi&m=icam
Отсюда вам необходимо скачать mariadb сервер и установить себе на компьютер. В моей программе идет один супер пользователь для упрощения. Пользователь root с паролем 5950. Однако вы можете задать свой пароль по желанию.
После скачивания и установки, создания пользователя вам необходимо откопировать мою базу данных, которая лежит в файле task_manager.sql

Вот пример команды для пользователя с root. mysql -u root -p task_manager < task_manager.sql
-p флаг значит пароль.
при установке mariadb server  mysql/mariadb взаимозаменяемы. 

2.https://mariadb.com/downloads/connectors/connectors-data-access/cpp-connector
 После установки сервера вам необходимо будет установить mariadbcpp connector
Вы его устанавливаете, а потом переходите в Visual Studio. Библиотека по умолчанию под MSVC. 
Далее вам надо будет подключить библиотеку.
https://www.youtube.com/watch?v=NLv8WV43TuA&t=48s
Можно взять пример с этого ролика.

3.https://github.com/wjakob/nanogui
 После установки библиотеки для подключения к базе данных, вам необходимо будет установить пакет библиотек для nanogui. Для вашего удобства я их скинул в архив CPP_Libraries, однако если что на странице гитхаб с nanogui даны ссылки на все cmake файлы для собственной сборки библиотечных файлов. mariadb connector версии 1.02. Его я не скинул, тк он много весит.

4. По такому же принципу подключаете и остальные файлы к проекту Microsoft Visual Studio.

После этого можете запускать программу.
